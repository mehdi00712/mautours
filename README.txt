
IslandVista Tours – Starter (Static HTML/CSS/JS)

Deploy
- Drop the folder on Netlify, Vercel (static), or GitHub Pages.

What’s included
- EN/FR toggle (edit js/lang/*.json)
- Excursions listing powered by window.TOURS (edit in excursions.html or move to a JSON file)
- Simple forms wired to placeholders – replace with EmailJS or your backend endpoint
- Responsive UI, dark theme, gradient accents
- Schema.org JSON-LD for better SEO
- WhatsApp quick chat button
